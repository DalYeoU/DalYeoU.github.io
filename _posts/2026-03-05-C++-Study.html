오늘 학습한 내용 정리

C++에 대해 학습한 내용 정리

오늘 C++에 대해 학습한 것들은 포인터, 동적할당, 구조체 이렇게 3가지입니다.

1. 포인터(Pointer)

포인터는 변수의 주소를 저장하는 변수입니다. 일반 변수는 데이터를 저장하지만 포인터는 변수의 메모리 주소를 저장합니다.

포인터 핵심 개념

  • 일반 변수 : 데이터를 직접 저장 (int a = 10;)
  • 포인터 변수 : 데이터가 저장된 메모리 주소를 저장 (int* p = &a;)
  • 주소 연산자&a : 변수의 메모리 주소를 반환
  • 역참조 연산자*p : 포인터가 가리키는 변수의 값을 반환

간단한 예제


#include 
using namespace std;

int main() {
    int num = 10;       // 일반 변수
    int* p = #      // 포인터 변수: num의 주소 저장

    cout << "num의 값: " << num << endl;        // 10
    cout << "num의 주소: " << &num << endl;     // 메모리 주소
    cout << "포인터 p의 값(주소): " << p << endl; // num의 주소
    cout << "포인터 p가 가리키는 값: " << *p << endl; // 10

    return 0;
}

포인터와 일반 변수의 차이

구분 일반 변수(int a = 10;) 포인터(int* p = &a;)
저장하는 것 값(데이터) 주소(메모리 주소)
접근 방법 변수 이름(a) *p로 값 접근
활용 예시 단순 값 저장 동적 메모리 관리, 배열,함수 전달

포인터는 잘 사용하면 효율적으로 데이터를 관리할 수 있는 변수 입니다.

포인터를 사용할때 주의할 점은 delete를 사용하여 해제하지 않으면 메모리 누수가 발생할 수 있습니다.

2. 동적할당(Dynamic Memory Allocation)

우선 메모리의 구조는 크게 1. 스택 메모리, 2. 힙 메모리로 나뉩니다.

스택 메모리

  • 할당 시점 : 컴파일 시점에 크기와 위치가 결정됨
  • 저장 대상 : 함수 내에서 선언된 지역 변수
  • 관리 방식 : 함수가 끝나면 자동으로 해제
  • 속도 : 빠름(메모리 접근이 단순)
  • 제한 : 크기 변경 불가, 공간이 제한적 → 큰 배열이나 동적 구조에는 부적합

힙 메모리

  • 동적 할당 : 실행중에 프로그래머가 요청해서 할당
  • 저장 대상 : 동적 할당된 객체, 배열, 자료구조
  • 관리 방식 : new / delete로 직접 관리
  • 속도 : 스택보다 느림 (메모리 요청/해제가 복잡)
  • 장점 : 크기를 실행 중에 결정 가능, 유연성 높음

간단한 예제


#include 
using namespace std;

int main() {
    int size;
    cout << "배열 크기 입력: ";
    cin >> size;

    // 힙 메모리에 동적 할당
    int* arr = new int[size];

    for (int i = 0; i < size; i++) {
        arr[i] = i + 1;
    }

    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }

    delete[] arr; // 반드시 해제
}

동적할당 또한 delete를 사용하여 메모리를 해제해야 합니다. 그렇지 않으면 메모리 누수가 발생할 수 있습니다.

3. 구조체(Structure)

구조체는 여러개의 변수를 하나로 묶어 관리하는 데이터 구조입니다.

만약 학번, 이름, 성적을 관리하는 배열을 만드려면 직접 변수를 선언해서 만드는 것 보다 구조체를 사용해서 관리하는게 훨씬 편리합니다.


#include 
using namespace std;

// 구조체 정의
struct Student {
    int id;         // 학번
    char name[20];  // 이름
    double grade;   // 성적
};

int main() {
    // 구조체 변수 선언 및 초기화
    Student s1 = {2023001, "Kim", 4.3};

    // 멤버 접근
    cout << "학번: " << s1.id << endl;
    cout << "이름: " << s1.name << endl;
    cout << "성적: " << s1.grade << endl;

    return 0;
}
